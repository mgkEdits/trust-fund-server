# Trust Fund Server (API)

## Overview
The REST API backend for the Trust Fund Welfare platform.

## Tech Stack
- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: PostgreSQL
- **Testing**: Jest & Supertest

## Getting Started
Prerequisites: ensure you are using the project `devbox` shell.

1. Install dependencies: `npm install`
2. Setup Env: Copy `.env.example` to `.env`
3. Run development server: `npm run dev`
4. Run tests: `npm test`
